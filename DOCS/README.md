# DOCS

Ovaj folder sadrzi moje beleke za tehnicki intervju: sta je uradjeno, zasto je izabran taj pristup, i koje su bile alternative.

## Sadrzaj
- 01-setup.md: inicijalna konfiguracija (Webpack, Babel, ESLint) i kako se pokrece aplikacija.
- 02-architecture.md: struktura foldera, routing i layout.
- 03-data-flow.md: GraphQL i Apollo Client, kako tece data flow.
- 04-ui-design.md: tematizacija, MUI komponente i vizuelna odluka.
- 05-alternatives.md: koje druge opcije su bile moguce.
- 06-presentation.md: govor za intervju, korak-po-korak objasnjenje.
